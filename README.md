# 📱 LAB 5 – Convertisseur Température & Distance

## 📝 Description
Cette application Android permet de convertir des valeurs de température et de distance à l’aide de deux onglets (Fragments).  
Elle offre une interface simple pour effectuer des conversions rapides entre différentes unités.

---

## 🎯 But
- Manipuler les Fragments et les Onglets (TabLayout + ViewPager2)
- Implémenter des conversions :
  - Température (°C ↔ °F)
  - Distance (Km ↔ Miles)
- Gérer :
  - Un menu avec option Quitter
  - L’interception du bouton Retour avec confirmation
